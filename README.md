# GPIJALANSUCI.OR.ID



Portal Web GPJ Jalan Suci https://www.gpijalansuci.or.id/

## Setup

Menggunakan Codeigniter 4 dengan HMVC.

### Clone 

```
git clone https://github.com/yusufmambrasar/gpijalansuci.git
```

### Set Writeable

```
sudo chmod -R 0777 writeable
```
